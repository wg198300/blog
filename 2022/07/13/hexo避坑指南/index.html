hexo避坑指南

发表于2022-07-13|更新于2022-08-26|tech
|阅读量:

hexo部署到github上就不 能显示主题样式
方法：
url：在根配置文件_config.yml中修改url：
theme: butterfly

例：
url：https://wg198300.github.io/blog/

添加国内镜像源
如果没有梯子的话，可以使用阿里的国内镜像进行加速。

npm config set registry https://registry.npm.taobao.org

GIT推送需要修改推送地址和推送方式

deploy:
type: git
repository: https://github.com/wg198300/blog.git
branch: main

github ssh创建秘钥

复制.PUB文件内容

推送，更新，清理集合命令
      hexo clean && hexo g && hexo d
建立新分类
hexo new  page categories
