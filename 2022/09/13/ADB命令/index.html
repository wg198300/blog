adb命令及手机设置

|阅读量:

adb及手机设置

  1. 安装adb

  2. 手机打开开发者选项，打开USB调试模式

  3. 查看手机连接情况 adb devices

无root权限卸载

打开命令行

获取APP安装包

第一张

  • adb shell pm list packages 所有安装包列表

  • adb shell pm list package [-s系统应用] [-3第三方应用] 区分系统APP和第三方APP

包太多找不到要卸载的APP的包，使用下面第二种方法

第二种

  1. 手机回到主页，退出手机APP后台

  2. 命令行输入命令 ：adb shell dumpsys window w |findstr / |findstr name=

  3. 打开要卸载的APP（如图为小米的系统更新），命令行点击enter键，查看Log

截图中标记的就是要卸载APP的包名

  1. 命令行输入命令： adb shell pm uninstall –user 0 [appName] 表示删除用户空间0的应用。这跟卸载普通应用是同一个方式 ，显示success卸载完成

附系统软件删除

有root权限

同 无root权限-第二种方式，获取到APP包名后，直接使用命令 adb uninstall [appName] 卸载

禁用应用

禁用应用可以防止应用后台运行，隐藏桌面图标，但占用空间不会减少，出现问题可随时启用。

例如：adb shell pm disable-user com.miui.systemAdSolution 禁用小米广告程序

禁用应用：adb shell pm disable-user 应用包名

开启应用：adb shell pm enable 应用包名
